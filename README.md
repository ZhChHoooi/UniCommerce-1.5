# UniCommerce-1.5
UniCommerce-1.5: A compact 1.5B-parameter unified e-commerce LLM built on Qwen2.5. By leveraging structured prompts and CoT distillation, it consolidates 6 core tasks (retrieval, recommendation, QA, etc.) into a single generative framework. Includes an evaluation suite balancing accuracy, latency, and cost for real-world pipelines.
